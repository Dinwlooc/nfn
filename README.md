
# 游戏项目简介
本项目使用 [Godot 引擎](https://godotengine.org/) 开发，目标为实现多人卡牌对战游戏。目前仍在开发阶段。
开发者属于业余爱好者，项目结构不具有参考价值。
项目的现行计划和架构说明暂不公开。
## 项目结构概览
- **core/Script**: 定义游戏运行所需的类与场景文件。
- **resource/**: 官方资源包文件夹。
- **mods/**: 外部资源包文件夹。
- **addons/**: Git 插件等。
## 主要功能模块
- **系统**：服务器端运行，管理游戏状态、事件处理和数据同步。
- **渲染**：客户端运行，负责游戏画面渲染和用户交互。
- **全局控制**： 提供调试指令、全局注册和配置管理等功能。
- **通信**：处理客户端与服务器之间的网络通信。
- **辅助系统**：包括场景转场和动画工具等。
## 开发环境与依赖
- **Godot 引擎**：项目使用 Godot 4.7.stable_mono 开发，请确保使用兼容版本打开。
- **Git 插件**：项目中集成了 `godot-git-plugin`，用于版本控制。
## 如何运行项目
1. 安装 [Godot 4.x](https://godotengine.org/download/)。
2. 克隆本仓库到本地。
3. 使用 Godot 打开项目目录并运行。
